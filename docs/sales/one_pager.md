# Lunchportalen — salgs-one-pager (én side)

## Hva det er

**Lunchportalen** er en vertikal SaaS-plattform for **bedriftsmat og lunsj** som samler **bestilling, drift, kjøkken og roller** med **vekst- og inntektslogikk**: AI-støttet innhold, kontrollert annonsering, sporbarhet og beslutningsstøtte — levert med **enterprise-tenancy** og **revisjon** der det er aktivert.

## Hvorfor det betyr noe

Matbedrifter og catering taper ofte margin på **fragmenterte verktøy**: markedsføring uten klar kobling til lønnsomhet, **annonser uten kontroll**, og **manuelle beslutninger** som ikke skalerer på tvers av lokasjoner. Lunchportalen reduserer friksjonen og gir **forutsigbarhet** — uten å love magi.

## Hva dere får

| Område | Leveranse |
|--------|-----------|
| **Drift** | Ordre, ukeplan, roller — én operativ sannhet |
| **Vekst** | AI-innhold og annonseflyt med godkjenning der det er bygget inn |
| **Økonomi** | Bedre sporbarhet mellom aktivitet, omsetning og margin (avhengig av data) |
| **Enterprise** | Rollebasert tilgang, audit trail, skalerbar multi-location-modell |

## For hvem

**Mid-market mat**, **catering** og **multi-location** som vil styre både **operasjon** og **vekst** i samme system.
