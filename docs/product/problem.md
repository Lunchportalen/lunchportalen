# Problembilde — hvorfor markedet trenger dette

## 1. Annonser koster penger uten tydelig kobling til margin

Mange bedrifter kjører annonser uten stabil kobling mellom **spend**, **kampanje** og **faktisk lønnsomhet per ordre**. Resultatet blir diskusjoner om «aktivitet» i stedet for om **kontrollert vekst**.

## 2. Lite sammenheng mellom markedsføring og resultat

Innhold publiseres i ett spor, tall i et annet. Uten felles operasjonelt system blir det vanskelig å svare på enkle spørsmål: *Hva skapte denne omsetningen? Hva kostet det? Hva gjør vi neste uke?*

## 3. Svake eller trege lager- og innkjøpsbeslutninger

Når etterspørsel, pris og innkjøp ikke sees i sammenheng, oppstår **enten** overkjøp og svinn **eller** tomme hyller og tapte ordrer. Beslutninger tas ofte manuelt og sent.

## 4. Manuelle, trege beslutningsløp

E-post, regneark og møter skalerer dårlig. Multi-location forsterker problemet: **ulike praksiser**, vanskelig å auditere, og høy avhengighet av enkeltpersoner.

## Konsekvens for enterprise-kjøpere

- **Risiko:** Manglende sporbarhet og roller ved revisjon eller avvik  
- **Kost:** Timer brukt på koordinering i stedet for forbedring  
- **Vekst:** Vanskelig å replikere «det som fungerer» på neste lokasjon  

## Bro til løsningen

Løsningen er ikke «mer verktøy», men **ett operativt lag** med mat, ordre, roller, innsikt og kontroll — se [solution.md](solution.md).
