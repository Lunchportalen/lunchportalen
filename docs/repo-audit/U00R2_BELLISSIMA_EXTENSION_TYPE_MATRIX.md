## Matrix Judgment
Lunchportalen has real Bellissima-inspired equivalents for sections, workspaces, workspace context, actions, views, footer apps, and management routes. The hard gaps are where Umbraco expects a true extension runtime, persisted Data Types / Document Types, entity bulk actions, and a platform-wide property editor schema/preset model. Those gaps are not naming gaps. They are real `STRUCTURAL_GAP` or `REPLATFORMING_GAP` gaps.
