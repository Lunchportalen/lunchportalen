## Import Graph Judgment
The import graph shows a real layered editor, but it also shows why the workspace still feels heavy: the code has multiple thin composition layers, yet the operational state still falls back into one giant orchestrator (`ContentWorkspace.tsx`) and several parallel discovery/preview surfaces.
