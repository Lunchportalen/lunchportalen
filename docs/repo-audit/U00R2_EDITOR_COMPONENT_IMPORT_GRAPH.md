## Import Graph Judgment
The import graph shows a real layered editor. It also shows why the workspace still feels heavy: there are several thin composition layers, but the operational truth still falls back into one giant orchestrator (`ContentWorkspace.tsx`) plus parallel discovery, preview, and governance surfaces that the user experiences as one system.
