# CMS — Control towers contextualization (CP7)

## Mål

- Behold **én control plane** der company admin, kitchen, driver og superadmin er **moduler** med tydelig «hva leses / hva skrives».

## CP7

- Forsterker **meny-kjeden** via native publish panel på `/backoffice/week-menu`.
- Ingen nye tower-systemer; ingen endring av operativ runtime-sannhet.

## Tekstlig standard

- Hver modul: **kilde** (server) + **safe action** (rute) + **påvirkning** (hvem).
