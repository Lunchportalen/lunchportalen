# CMS — Week / menu UI truthfulness

**Prinsipp:** UI skal ikke antyde at `weekPlan` styrer employee week.

## CP4-tiltak

- **Grønt panel:** operativ publish-kjede (meny + `/api/week`).
- **Amber panel + banner:** `weekPlan` = editorial-only.
- **Toppstripe:** LIVE moduler viser **Published** som menneskelesbar etikett (teknisk badge fortsatt LIVE i data).

## Forskjell content vs meny

- Content pages: Postgres publish.
- Meny: Sanity-dokumenter — forklart i publish-panelet.
