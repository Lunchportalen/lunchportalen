# SEO / Growth — visuell runtime (2D2)

---

## 1. Design system (2A)

- **Max bredde:** `max-w-[1440px]`.
- **Topp luft:** `pt-[27px]` på flaten.
- **Primær CTA:** «Analyser side» — hot-pink-akzent (`border` / lett bakgrunn) i tråd med én tydelig primær handling.
- **Score:** fargekodet tekst (grønn / amber / rød) basert på terskel.
- **Kort:** `rounded-2xl`, border `lp-border`, rolig typografi (`font-heading` for overskrifter).
