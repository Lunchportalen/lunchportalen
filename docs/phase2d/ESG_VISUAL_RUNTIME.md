# ESG — visuell flate (Phase 2D3)

## Designvalg

- **Bakgrunn:** `lp-bg` / kort med `lp-border`, hvit/krom — samme som annet backoffice-innhold.
- **Accent:** én tydelig interaksjon (firmavalgt tilstand med diskret `lp-hotpink` ring); primær lenke som `rounded-full` border-knapp (ikke hele flater i rosa).
- **Tone:** seriøs, datadrevet — ingen «grønnvask»-illustrasjoner; bruk av **amber** kun for «mangler data»-banner.
- **Typografi:** én H1 per side; hierarki med `text-sm`/`text-xs` for metode og KPI.

## Unngått

- Nye globale shells eller header — bruker eksisterende `BackofficeShell` + `TopBar`.
