# U21 — Workspace actions runtime

## Primær / sekundær

- **Primær:** `toolbar` — `role="toolbar"` aria-label «Primære handlinger».
- **Sekundær:** `secondaryActions` — egen verktøylinje.

## Innhold (eksempler)

- **Domener:** Runtime, Uke & meny, Content + Superadmin (sekundær, mutasjon).
- **Uke & meny:** Sanity Studio, Runtime, Content + Domener (sekundær).

## Footer apps

- **`footerApps`:** region under innhold (fullBleed: nederst i scroll-kolonne; default: etter `children`).
