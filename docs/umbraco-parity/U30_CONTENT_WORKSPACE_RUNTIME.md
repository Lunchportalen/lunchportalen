# U30 — Content workspace runtime

## Layout

| Komponent | Endring |
|-----------|---------|
| `SectionShell` | Tree: **340–480px** |
| `ContentWorkspaceWorkspaceShell` | Tri-pane: **260–360 | 1fr | 320–420** |

- Én tydelig status/handling i topp (`ContentTopbar`), ikke gjentatt i editor-stripe.
