# U30 — Action hierarchy model

## Primær (alltid synlig, store mål)

- **Publiser** / **Sett til kladd** (`ContentTopbar`)
- **Lagre** / **Lagre og forhåndsvis** (`ContentSaveBar`)

## Sekundær (redaksjonell / recovery)

- Kopier support, Lagre på nytt, Last siden på nytt
- Åpne offentlig side (når tilgjengelig)

## Lokale editor-handlinger

- Rediger ↔ Forhåndsvisning, enhetsvalg
- Blokk: flytt, dupliser, legg til

## Review / governance

- Ikke blandes inn i primær rad — ligger i egne paneler / settings / governance-sider.

## Styling-prinsipp

- Én hot-pink-aksent for aktiv modus (allerede i TopBar/modus).
- Grønn primær kun for Publiser (allerede).
