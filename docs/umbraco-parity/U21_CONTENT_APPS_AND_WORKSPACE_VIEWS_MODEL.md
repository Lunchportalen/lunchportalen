# U21 — Content apps og workspace views

## I dag

- **Content:** stort workspace med paneler/rails (eksisterende).
- **Media:** klient-side bibliotek med `BackofficeWorkspaceHeader`.
- **SEO/Social/ESG:** `CmsGrowthModuleCallout` + runtime-klienter i `fullBleed`.
- **Week/menu:** orchestrator-paneler + tabeller.
- **Domains:** domain action cards + tabeller.

## CP21 / U21 standardisering

- Felles **kontekst-blokk** (`contextSummary` + `statusChips`) der redaktør må skille CMS vs runtime.
- `publishHistoryNote` beholdes for ærlig historikk-fortelling.
