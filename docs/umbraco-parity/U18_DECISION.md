# U18 — Decision

**Dato:** 2026-03-29

## 1. Endelig beslutning

**GO WITH CONDITIONS**

- **GO:** U18 leverer **sterkere global discovery** (alias + tårn i palett), **tydeligere historikk-/publish-fortelling** (ærlig strip), og **AI Control Center** med full modulposture-oversikt — uten ny søkemotor, historikkmotor eller orchestrator.
- **Betingelse:** Ekte global fulltext og én teknisk tidslinje krever **egen plattform** — se `U18_REPLATFORMING_GAPS.md`.

## 2. Hva som er oppnådd

- **CMS som hovedbase:** Backoffice + registry + palett + strips + AI Center under én kontrollfortelling.
- **Domener:** Uendret dataeierskap; **forklaring** og **lenker** forsterket (historikk, tårn, growth).
- **Ukemeny/ukeplan:** Uendret operativ kjede; Uke & meny nevnt eksplisitt i historikk-strip.
- **Discovery:** Ctrl/⌘K palett med alias-søk og runtime-tårn.
- **Historikk:** Punktliste per kilde + rollback-ærlighet.
- **AI control center:** `/backoffice/ai-control` med governance-tabell fra `MODULE_LIVE_POSTURE_REGISTRY`.
- **Sections/trees/workspaces:** Uendret modell; nye rader kun i **samme** `ENTRIES`.

## 3. Hva som fortsatt er svakt

- Ingen indeksert enterprise-søk.
- Social/worker fortsatt LIMITED/STUB der definert.
- Full workspace-toolbar-konsistens ikke tvunget på alle sider.

## 4. Nærhet til Umbraco / verdensklasse

**Sterkere** redaktør-OS-følelse på discovery og governance-**lesbarhet**; **ikke** teknisk identitet med Umbraco Cloud.

## 5. Før ubetinget enterprise-live-ready

1. Beslutning om **søkeprodukt** eller permanent palett-scope.
2. Eventuell **historikk-aggregator** — egen fase.
3. **AI kost/modell** UI — når eier prioriterer.

## 6. Kan vente

- Full toolbar-standard på alle workspaces.
- JSON export av registry.
