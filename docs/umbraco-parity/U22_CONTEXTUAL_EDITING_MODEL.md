# U22 — Contextual editing model

## Eksisterende (U21)

- `WorkspaceContextChrome`, primær/sekundær toolbar, footer apps på nøkkelflater.

## U22

- **Media:** collection toolbar forklarer at valg + bulk kun påvirker utklippstavle.
- **Blokkeditor:** property dataset-tabell forklarer **hva som er schema-styrt** vs rå JSON (avansert).

## Standardisering

- Samme **toolbar**-komponent der collection-mønster brukes (media, vekst-lister).

## «Hva redigerer / publiserer / påvirker»

- Uendret runtime-sannhet — tydeligere **feltnivå** i modal via dataset-explainer.
