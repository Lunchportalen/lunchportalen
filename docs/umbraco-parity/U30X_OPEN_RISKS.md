# U30X — Open risks

1. **DB-sannhet:** Lokale/migrerte miljøer kan fortsatt avvike — degradert UI må ikke maskeres som «tomt innhold» uten at redaktør forstår hvorfor.
2. **Inspector:** Stor `ContentWorkspacePropertiesRail` — risiko for fortsatt opplevd «tett» UI til tross for ytre shell-justeringer.
3. **Dokumentasjonsdrift:** Mange `Uxx`/`CPxx` filer — krever eierstyrt konsolidering senere.
4. **Performance:** Større preview kan øke layout-kost — overvåk på svake klienter.
