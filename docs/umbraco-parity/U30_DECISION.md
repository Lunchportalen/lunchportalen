# U30 — Decision

**GO WITH CONDITIONS**

Full Umbraco 17 editor-paritet krever videre arbeid på inspector-feltgrupper og ev. rolleutvidelse for tree API — utenfor denne leveransen.

## Levert

- Audit-logg robust ved manglende tabell.
- Tree-feil forklart for 401/403.
- Content workspace: bredere tree/tri-pane, større preview, færre dupliserte statusrader.

## Betingelser

- Legg ved piksel-bevis i `U30_EDITOR_FAILURES_FROM_SCREENSHOT.md` når skjermdump finnes.
- Superadmin-only tree API er dokumentert forventet oppførsel — ikke en «bug» uten produktbeslutning.
