# CP10 — Open risks

1. **Forventningsglidning:** Brukere kan tro at ⌘K søker i **innhold**, ikke bare **modulnavn** — paletten sier eksplisitt «kun navigasjon».
2. **Nav-drift:** Nye `/backoffice/*` sider må legges i `BACKOFFICE_NAV_ITEMS` for palett-paritet.
3. **Unified history:** Risiko for produktkrav om én tidslinje — **må** holdes innenfor faktiske API-er.
4. **Indeks-prosjekt:** Hvis «global søk» bestilles, unngå duplikat indeks parallel med Sanity Studio.
