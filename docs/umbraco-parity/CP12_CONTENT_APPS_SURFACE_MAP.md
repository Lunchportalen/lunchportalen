# CP12 — Content apps surface map

**Dato:** 2026-03-29

## Finnes i dag

| Flate | Overflate |
|-------|-----------|
| Content | Tre, editor, paneler (SEO, CRO, recovery, AI, …) |
| Media | Bibliotek, metadata |
| Domener/kunder/avtale | Kort, tabeller, action surfaces |
| Uke/meny | Orkestrator, publish-paneler, tabell |
| SEO/Social/ESG | Callout + klient |

## CP12-forbedring

- **Global historikk-strip** gir tverr-kontekst uten ny app.
- Palett inkluderer **papirkurv** og **control tower** for discovery.

## Mangler (relativt Umbraco)

- Dedikert **Versions**-app per entitet — krever dypere backend eller Studio.

## Grenser

- Ingen nye parallelle editorer.
