# U19 — Decision

**Dato:** 2026-03-29

## 1. Endelig beslutning

**GO WITH CONDITIONS**

- **GO:** U19 leverer **indeksert discovery-rangering** (precomputert manifest-indeks), **redaksjonell tidslinje-UX** (tre spor), og **utvidet AI governance UI** — uten ny søkemotorplattform, uten aggregator-tidslinje-API, uten ny orchestrator.
- **Betingelse:** Ekte global fulltext og én teknisk tverrkilde-tidslinje krever **egen plattform** — se `U19_REPLATFORMING_GAPS.md`.

## 2. Hva som er oppnådd

- **CMS som hovedbase:** Uendret; discovery og historikk/AI forsterket i kontrollflaten.
- **Domener:** Uendret eierskap; forklaring og navigasjon sterkere.
- **Uke/meny:** Uendret operativ kjede; spor C i tidslinje-UX.
- **Discovery:** Palett med **rankering** etter indeksert blob.
- **Historikk:** Tre-spors **visuell** fortelling + eksisterende tekst.
- **AI control center:** Modulposture + human/kost/konfig-paneler.
- **Sections/trees/workspaces:** Fortsatt `BACKOFFICE_EXTENSION_REGISTRY` som eneste manifest.

## 3. Hva som fortsatt er svakt

- Ekstern indeks (Elasticsearch m.m.).
- Sammenslått event feed.
- Interaktiv AI-modell/kost UI.

## 4. Nærhet til Umbraco / verdensklasse

**Sterkere** «quick find»-følelse og **tydeligere** editorial story; **ikke** teknisk identitet med Umbraco Cloud Search.

## 5. Før ubettinget enterprise-live-ready

1. Produktvalg: **enterprise søk** eller permanent indeks-light.
2. Eventuell **audit-aggregator**.
3. **Usage metering** for AI.

## 6. Kan vente

- Full toolbar-paritet på alle workspaces.
