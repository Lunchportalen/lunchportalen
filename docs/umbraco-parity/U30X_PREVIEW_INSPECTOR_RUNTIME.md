# U30X — Preview + inspector runtime

## Preview

- Fjernet overflødig toppmerkelapp.
- Økt padding og `shadow-sm` på preview-kort.
- Bredde: `max-w-xl` (standard), `max-w-3xl` når visual inline edit er på.

## Inspector

- `RightPanel`: overskrift «Inspektør»; litt tettere fane-knapper.
- Dypere gruppering (innhold/design/SEO/…) ligger fortsatt i `ContentWorkspacePropertiesRail` + faner — ikke full omorganisering i U30X.
