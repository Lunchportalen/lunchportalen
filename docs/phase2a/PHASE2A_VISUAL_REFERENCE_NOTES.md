# Phase 2A V3 — visual reference notes

## GetInspired (stemning, ikke 1:1)

- **Hentet:** energisk kommersiell klarhet, «premium retail»-rytme, kampanjehierarki som *struktur* (tydelige nivåer), moderne fart uten støy.
- **Oversatt:** skarpere panel-hierarki, glass på command-flater, mer «scanbar» CMS uten nye farger utenom aksent-disiplin.

## Stormberg (stemning)

- **Hentet:** tillit, robusthet, enkel skannbarhet, jordnær funksjonell UI.
- **Oversatt:** rolige bakgrunner, tydelige rammer, begrenset motion, ingen candy-gradient på alt.

## Lunchportalen egen identitet

- Varm cream base (`--lp-bg`), **én** hot-pink/neon-aksel, gull støttet der allerede i merkevare.
- Enterprise RC-modus: forutsigbarhet > effektjakt.

## Neon rosa — kontrollert

- TopBar aktiv fane, focus rings, AI-introkort (subtil rosa kant), ikke tabeller eller lange tekster.

## Glass — hvor / hvor ikke

| Glass OK | Ikke glass |
|----------|------------|
| TopBar, targeting-bar, AI-rail intro, dialoger | Data-tabeller, lange skjema, kritisk drift |
