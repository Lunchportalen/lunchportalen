# Commercial One-Pager

**Status:** Standardmodell for enterprise-dialog — **ikke** juridisk kontrakt. Avvik kun skriftlig avtalt.

## Hvorfor Lunchportalen
- Kontroll og forutsigbarhet i leveranser.
- Mindre matsvinn via tydelige cutoff-regler.
- Lav administrasjonsbelastning for konsern og HR.

## Standard kommersiell modell
- 12 mnd binding
- 3 mnd oppsigelse
- Minimum 20 ansatte
- Standard modell uten unntak (ingen parallell «unntaksflate» i standardtilbud — tilpasninger avtales eksplisitt)
